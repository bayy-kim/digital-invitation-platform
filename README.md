## 📖 Overview

Digital Invitation Platform adalah platform **SaaS (Software as a Service)** dan **E-Commerce** untuk pembuatan undangan digital modern.

Platform ini memungkinkan pengguna membuat, mengelola, mempublikasikan, dan membagikan undangan digital melalui Builder berbasis visual (WYSIWYG), lengkap dengan sistem pembayaran online, dashboard analitik, program reseller, serta manajemen multi-tenant.

Repository ini digunakan sebagai pusat dokumentasi dan pengembangan seluruh sistem.

# ✨ Key Features

- Multi Tenant SaaS

- Admin Dashboard

- Google OAuth Login

- Guest Book
